# netflix-analysis

## About
Exploratory Data Analysis of Netflix Movies and TV Shows dataset (8807 titles).

## Key Findings
- Movies dominate Netflix content (70% vs 30% TV Shows)
- USA produces the most content
- Most content is added to Netflix in the same year as release
- TV-MA is the most common rating

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scipy (statistical analysis)

## Dataset
[Netflix Movies and TV Shows](https://www.kaggle.com/datasets/shivamb/netflix-shows)
